We will be using the default Swagger API provided by AWS for testing purposes.
A small backend is provided by AWS for testing so that beginners can concentrate on learning API Gateway without worrying about creating a back-end first while in the initial days.

Default Backend URI:
http://petstore.execute-api.us-east-1.amazonaws.com/petstore/pets
